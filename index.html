Search & Prompt Ideas

Creative ways to search when you want better summaries or context

1. Search by role “Explain this dialogue as if you are a teacher / doctor / journalist.”
2. Search by goal “Summarize focusing on action items / decisions / emotional tone.”
3. Time-based summary “Short summary (1–2 lines) / Medium (3–5 lines) / Detailed (bullet points).”
4. Ask for examples “Provide sample follow-up questions or clarifying prompts.”
5. Combine sources “Summarize and compare with X (e.g., meeting notes, chat history).”
Tip Press Ctrl/Cmd + Enter to submit quickly.
